# Demo App for Pointotech React Native Combobox

## Set up and run on Android

```
yarn global add react-native-cli@latest
yarn
npx react-native run-android
```

## Set up and run on iOS

```
yarn global add react-native-cli@latest
yarn
cd ios
rm -rf Pods
rm Podfile.lock
pod install
cd ..
npx react-native run-ios
```
